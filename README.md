### 📁 Featured Projects

*   **[AirScience: Flight Ticket Price Prediction](https://github.com/AnhThy3447/AirScience_DS108)** ✈️

An end-to-end ML system predicting flight prices in Vietnam using a custom Selenium scraping pipeline (46k+ records) and Gradient Boosting. Designed a dynamic "Buy/Wait" recommendation engine.
*   **[IT Job Market Analysis (HCMC & Hanoi)](https://github.com/AnhThy3447/itviec_job_market_analysis_IS403)** 💼

Built an analytical and forecasting solution to map tech recruitment trends across Vietnam's two major tech hubs. The project identifies shifting tech stack demands and discovers hidden skill co-occurrences, providing valuable market insights for both tech job seekers and recruiters.

*   **[Restaurant Review Sentiment & Aspect Analysis](https://github.com/AnhThy3447/Review_sentiment_analysis_DS304)** 🍽️

Developed a fine-grained aspect-based sentiment analysis (ABSA) system to help restaurants automatically extract and evaluate customer feedback. Instead of just rating a review as overall positive or negative, the system breaks down comments into specific business dimensions (Price, Quality, Environment, Service) and measures the exact satisfaction level for each aspect, enabling managers to pinpoint precise areas for operational improvement.

*   **[Vietnamese Narrative Coreference Resolution](https://github.com/dbthyy/viet_bio_coref)** 📖

Built an end-to-end NLP pipeline to solve coreference resolution in Vietnamese narrative texts. The system addresses the challenge of tracking characters and entities across complex stories by combining a Transformer-based mention detection module (BIO tagging) with a pairwise classification model and Union-Find clustering, effectively linking scattered entity mentions into unified, coherent identity chains.
